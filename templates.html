This toolkit provides information on, and specifications about, the equipment used and available in the Lightbox Gallery.



The Lightbox Gallery is an open ended experimental space in the Harvard Art Museums dedicated to projects and artworks that use new media and emerging technologies to explore the museums' collection. If you can write code for the web, you can build projects to run in the Lightbox. Here are a few template projects to get you started.
